# 🌱 Sweet Farm – 2D Sugarcane Farming Simulation Game

A 2D sugarcane farming simulation game built with Unity and C#, featuring crop growth, watering mechanics, pest management, animations, and interactive farming gameplay.

---

## 📖 Description

Sweet Farm is an interactive **2D sugarcane farming simulation game** developed using **Unity Engine** and **C#**. The game provides an engaging and educational experience by simulating the real-world process of sugarcane cultivation.

Players take on the role of a farmer and perform essential agricultural activities such as planting sugarcane, watering crops, removing harmful pests, and monitoring crop growth. The game incorporates multiple growth stages of sugarcane, touch-based interactions, animations, and skill-check mechanics to create an immersive farming experience.

The project demonstrates the use of Unity's 2D game development features, including sprites, animations, prefabs, scene management, and mobile touch controls.
